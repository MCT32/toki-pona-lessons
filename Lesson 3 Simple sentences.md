## Sentence structure
<style>
@font-face {
  font-family: 'nasin nanpa';
  src: url('./res/nasin-nanpa-4.0.2-UCSUR.otf') format('truetype');
  font-weight: normal;
  font-style: normal;
}
</style>
- Same word order as English: noun, verb, object.
- Absolutely no conjugations.
- Particle **li** (<span>󱤧</span>) comes before the verb.
<!-- element style="font-family:nasin nanpa;" -->
- Particle **e** (<span>󱤉</span>) comes before the object.
<!-- element style="font-family:nasin nanpa;" -->
